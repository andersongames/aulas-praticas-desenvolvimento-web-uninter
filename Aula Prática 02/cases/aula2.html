Agora que já construimos toda a estrutura e formatação do site que estamos desenvolvendo para a Doce e Dança, começaremos a criar as animações visuais. Todas as funções Javascript que iremos criar através da biblioteca jQuery estarão localizadas em um arquivo, scripts.js.

Exercício 2

Ainda utilizando o conceito visto nos slides 12-13 de jQuery, realize a ligação do nosso arquivo de funções, script.js ao nosso projeto.


Exercício 3

Com as duas tags acima definidas, agora precisamos incluí-las à nossa estrutura básica. Utilizando o conceito visto nos slides 12-13 de jQuery, defina a estrutura básica das nossas páginas, incluindo os arquivos de CSS, a biblioteca jQuery e o nosso arquivo de funções

Não se esqueça que os arquivos .css estão dentro da pasta css, e os arquivos .js estão dentro da pasta js!


Exercício 4

Agora, iremos criar os scripts que serão realizados quando a página for carregada. Relembrando o conteúdo visto nos slides 14-15 e 17-18 de jQuery, defina a sintaxe para criar as funções dentro do evento de carregamento da página.


Exercício 5

Para a função acima funcionar corretamente, ela precisa estar dentro de uma tag <script>, geralmente localizada antes da tag de fechamento do <body> - isso é utillizado como uma boa prática para que a página carregue mais rapidamente.

Ainda utilizando o conteúdo visto nos slides 14-15 e 17-18 de jQuery, defina novamente a estrutura básica da nossa página, colocando antes do final do <body> uma tag <script> com a função jQuery que indica que os scripts serão executados quando a página estiver pronta.


Exercício 6

Vamos fazer uma função teste, imprimindo no console do navegador uma variável. Aplicando o conteúdo visto nos slides 14-16 de jQuery, defina uma variável, teste recebendo a string "Olá mundo, no console do navegador!", e, quando o documento estiver pronto, imprima esta variável utilizando a função console.log.


Exercício 7

Baseado nos conteúdos apresentados nos slides 14-26 de jQuery, crie uma função que adicione o texto "Seja bem vindo" ao #header da página, quando esta estiver pronta.


Exercício 8

Quando o usuário clicar no botão para enviar o formulário de contato, devemos mostrar para ele uma mensagem que irá indicar que a mensagem dele foi enviada com sucesso.

Aplicando os conceitos vistos nos slides 14-26 de jQuery, crie uma função para que, quando houver o clique no botão #enviaForm, um alerta seja mostrado com a mensagem
"Sua mensagem foi enviada com sucesso, obrigado!".


Exercício 9

Iremos realizar uma animação na tela de Receitas - quando o usuário clicar na foto da receita, que possui a classe imgReceita, o texto descritivo, descricaoReceita, deverá deslizar de cima para baixo; e quando clicarmos novamente, ele deverá deslizar de baixo para cima.

Ainda utilizando os conceitos vistos nos slides 14-26 de jQuery, crie uma função que irá fazer com que essa animação seja realizada com sucesso.

Lembre-se: o método slideDown apenas faz com que ele deslize para baixo, e o método slideUp apenas faz com que ele deslize para cima.


Exercício 10

Complementando a animação criada no exercício anterior, quando a imagem for clicada, devemos adicionar uma borda, amarela (#ffff00), para passar o destaque de que é ela que está selecionada.

Ainda utilizando os conceitos vistos nos slides 14-26 de jQuery, complemente a função anterior para fazer com que essa animação seja realizada com sucesso.

Quando o objeto alvo da animação é o mesmo que conduz o evento, utilizamos $(this), para não repetir o seletor.
