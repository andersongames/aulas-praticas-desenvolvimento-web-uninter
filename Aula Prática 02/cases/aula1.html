Exercícios - HTML

Olá, nestes exercícios iremos criar uma página web para a empresa Doce e Dança.

Essa empresa, uma confeitaria, procurou você como desenvolvedor web para criar um site.

O site será simples, institucional e terá um formulário para contato.

Exercício 3

Exercício 4

A tabela de preços que criamos também deverá ser formatada. Ela deverá receber a formatação de 2 classes - tabela e borda-tabela, pois, há outras tabelas sem bordas no site.

Para a classe tabela, devemos definir que ela ocupará sempre o tamanho total do seu parent. A classe borda-tabela definírá a borda da tabela, que será sólida, com 1px de largura e com a cor #ccc (tom de cinza em hexadecimal).

Exercício 5

Em cada página do site que estamos desenvolvendo, o container que irá envolver o conteúdo será único - não irá se repetir. Portanto, podemos definir um ID para ele. A <div> que irá envolver o conteúdo será #conteudo. Ela terá as seguintes propriedades:

Padding de 10px, margem de baixo de 20px, uma borda sólida, de 1px e de cor #ccc no final, e um altura mínima de 30% em relação à página.

Aplicando os conceitos dos slides 4-10 de CSS, defina as propriedades ao ID definido.

Ainda com o conceito dos slides 4-10 de CSS e o conceito de herança, aplique uma formatação CSS definindo essa exigência do nosso contratante.


Exercício 7

Ainda utilizando o conceito de herança em CSS, defina que as imagens da <div id="conteudo"> possuam uma largura mínima de 40% e uma largura máxima de 80% em relação ao seu parent - definindo a largura de uma imagem, a altura será ajustada automaticamente.
